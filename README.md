# parkerdifuntorum.com — Image 1 Layout

This version matches the selected mockup direction:
- Large name-first hero
- Credentials and tagline below name
- All emblems grouped together below the hero text/info
- UF emblem appears before Sacramento State
- Uses attached real image assets for:
  - University of Florida
  - Sacramento State
  - U.S. Coast Guard
- Includes styled organization marks for:
  - PG&E
  - Liberty Utilities
  - Intel Corporation
  - Apple
  - General Electric

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/uf-emblem.png`
- `assets/sac-state-emblem.png`
- `assets/uscg-emblem.png`

## Notes
The PG&E, Liberty, Intel, Apple, and GE marks are stylized text/CSS representations so this package does not redistribute third-party trademark image files. You can replace them with official logo image files later by adding them to `assets/` and updating the emblem strip in `index.html`.

## Deploy
Upload all files to the root of the GitHub repository connected to Vercel.
